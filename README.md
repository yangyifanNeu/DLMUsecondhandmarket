## Project setup
```bash
安装前端依赖
npm install
```

```bash
进入到server文件夹
cd server
```

```bash
安装server端依赖
npm install
```



### Compiles and hot-reloads for development

```
启动前端代码
npm run serve
```

### Compiles and minifies for production
```
前端代码构建
npm run build
```

### Lints and fixes files
```
npm run lint
```
