# secondhand-market 校园二手市场 

## 前端技术栈

前端采用vue，vue-router，vuex，控件库使用ant design

## 后端技术栈

express，redis， sequelize

## Project setup
```bash
安装前端依赖
npm install
```

```bash
进入到server文件夹
cd server
```

```bash
安装server端依赖
npm install
```



### Compiles and hot-reloads for development

```
启动前端代码
npm run serve
```

### Compiles and minifies for production
```
前端代码构建
npm run build
```

### Lints and fixes files
```
npm run lint
```
